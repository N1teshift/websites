## Quick Lookup

### Find PD for an Objective

```typescript
import { getPDForObjective } from "@progressReport/student-data/config/pdKdMappings";

const pd = getPDForObjective("9Ae.01"); // Returns: "PD4"
```

### Find Objectives for PD

```typescript
import { getObjectivesForPD } from "@progressReport/student-data/config/pdKdMappings";

const objectives = getObjectivesForPD("PD1"); // Returns: ["9Ni.01", "9Ni.04"]
```

### Get All PDs

```typescript
import { getAllPDs } from "@progressReport/student-data/config/pdKdMappings";

const allPDs = getAllPDs(); // Returns: ["PD1", "PD2", ..., "PD54"]
```
